# 🌿 Plant Disease Detection Web Application

An AI-powered web application that detects plant diseases from leaf images and
provides treatment recommendations in **English** and **Odia (ଓଡ଼ିଆ)**.

---

## 📁 Folder Structure

```
plant_disease_app/
│
├── app.py                  ← Flask application (main entry point)
├── train_model.py          ← CNN training script (MobileNetV2 + PlantVillage)
├── demo_model.py           ← Quick demo model (random weights, for UI testing)
├── requirements.txt        ← Python dependencies
│
├── models/
│   ├── plant_model.h5      ← Saved Keras model (generated by training)
```

## 🚀 Quick Start

### 1. Install dependencies

```bash
cd plant_disease_app
pip install -r requirements.txt
```

### 2a. Option A — Quick test (demo model, no training needed)

This creates a model with **random weights** just to test the full pipeline:

```bash
python demo_model.py
python app.py
```

Open http://127.0.0.1:5000 in your browser.

### 2b. Option B — Full training (recommended for real use)

This downloads the PlantVillage dataset (~800 MB) and trains MobileNetV2:

```bash
python train_model.py
python app.py
```

Training takes ~30–60 minutes on GPU, longer on CPU.

## 🔌 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET  | `/`           | Serve the web UI |
| POST | `/predict`    | Predict disease from uploaded image |
| GET  | `/health`     | Server + model health check |
| GET  | `/classes`    | List all supported disease classes |

### POST /predict — Request

```
Content-Type: multipart/form-data
- file : image file (PNG/JPG/JPEG/BMP/WEBP, max 16 MB)
- lang : 'en' or 'od'
```

## 🧠 Model Architecture

- **Base**: MobileNetV2 (pre-trained on ImageNet)
- **Head**: GlobalAveragePooling → Dense(256) → Dropout(0.4) → Dense(128) → Dropout(0.3) → Softmax(38)
- **Training**: 2-phase — feature extraction then fine-tuning
- **Input size**: 224 × 224 × 3
- **Dataset**: PlantVillage (auto-downloaded via `tensorflow-datasets`)

---

## ⚙️ Configuration

Edit constants at the top of `app.py` or `train_model.py`:

| Variable | Default | Description |
|----------|---------|-------------|
| `IMG_SIZE` | 224 | Input image size |
| `BATCH_SIZE` | 32 | Training batch size |
| `EPOCHS_FT` | 10 | Feature-extraction epochs |
| `EPOCHS_FIN` | 5 | Fine-tuning epochs |
| `PORT` | 5000 | Flask server port |

## 🐛 Troubleshooting

**Model not found error:**
```
Run python demo_model.py (quick) or python train_model.py (full)
```

**TensorFlow installation issues (Apple Silicon / older GPU):**
```bash
pip install tensorflow-macos   # macOS with Apple Silicon
pip install tensorflow-cpu     # CPU only
```

**Port already in use:**
```bash
# Linux/Mac
lsof -i :5000 | grep LISTEN | awk '{print $2}' | xargs kill

# Windows
netstat -ano | findstr :5000
taskkill /PID <pid> /F
```
